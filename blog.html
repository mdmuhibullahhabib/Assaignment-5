Important FAQ

What is the document object Model (DOM)?

The DOM is a hierarchical representation of a web document, allowing dynamic access and manipulation via JavaScript.

How do you select an element from the DOM?

You can select DOM elements using methods like getElementById, querySelector, getElementsByClassName, or getElementsByTagName.

What is even delegation in the context of the DOM?and why is it useful.

Event delegation allows handling events on parent elements, improving performance and efficiency by reducing individual event listeners.

How do you manipulate an element a attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute, and change styles directly via the style property in the DOM.
